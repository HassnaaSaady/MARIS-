# MARIS-
## Maritime Navigation AI System

MARIS is a Big Data Maritime Intelligence Platform for real-time AIS vessel tracking, anomaly detection, collision-risk alerts, traffic-density analytics, and historical replay using Kafka, Spark, Delta Lake, PostgreSQL, FastAPI, React, Streamlit, and Machine Learning.
